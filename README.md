# CFA Level I 単語帳アプリ

CFA Level I 試験対策用の日本語単語帳モバイルアプリです。

## 機能

### 学習モード
- **384語**の用語一覧（10分野）
- 検索・フィルター機能
- 詳細画面（定義・Key Points・例文・公式・Pitfall・関連語）
- SRS（間隔反復学習）による復習機能
- 忘却曲線ダッシュボード

### クイズモード
- 四択問題（日本語→英語、英語→日本語）
- 入力式問題
- 穴埋め問題
- 概念説明クイズ
- 結果画面と苦手語の自動抽出

### RPG風ミニゲーム（CFA Quest）
- カードバトルシステム
- 10ステージ（各科目対応）
- ボス戦
- カード収集・強化・合成
- デイリーミッション
- ショップ・アイテムシステム
- CFA実問ストック（12問）

### 分野別用語数（合計384語）
| 分野 | 用語数 |
|------|--------|
| ETH（倫理・職業行為基準） | 10 |
| QM（定量分析） | 20 |
| ECON（経済学） | 20 |
| FSA（財務諸表分析） | 30 |
| CI（コーポレート・イシュアーズ） | 20 |
| **EQ（株式投資）** | **205** |
| FI（債券） | 25 |
| DER（デリバティブ） | 25 |
| AI（オルタナティブ投資） | 10 |
| PM（ポートフォリオ管理） | 19 |

## 技術スタック

- **フレームワーク**: React Native + Expo SDK 54
- **言語**: TypeScript
- **スタイリング**: NativeWind (Tailwind CSS)
- **状態管理**: React Context + AsyncStorage
- **テスト**: Vitest

## セットアップ

```bash
# 依存関係のインストール
pnpm install

# 開発サーバーの起動
pnpm dev

# テストの実行
pnpm test

# TypeScriptの型チェック
pnpm check
```
