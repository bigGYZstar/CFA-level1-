# CFA Level I 単語帳アプリ - UI設計書

## 概要
CFA Level I試験対策用の日本語単語帳アプリ。200語から1000語まで拡張可能。

## スクリーン一覧

### 1. ホーム画面 (Home)
- 学習進捗サマリー（今日の復習数、習得率）
- クイックアクセスボタン（学習モード、クイズモード）
- 今日の復習キュー表示

### 2. 学習モード - 一覧画面 (StudyList)
- 縦積みリストレイアウト（参考画像準拠）
- 左側に番号バッジ（薄いパステルカラー帯）
- 英語見出し（太字）+ 日本語見出し
- フィルター（科目/タグ/未学習/要復習/苦手）
- 検索バー
- 並び替え（頻度順/追加日/復習期限順）

### 3. 学習モード - 詳細画面 (TermDetail)
- 【日本語見出し（English Term / Abbrev）】
- 【定義】2-3行
- 【Key Points】最大2点
- 【CFA文脈 例文EN】1文（該当語ハイライト）
- 【例文JP】1文
- 【Pitfall】混同ペアを赤で明示
- 【関連語】TERM ID付き、ワンタップ遷移

### 4. SRS復習画面 (Review)
- フラッシュカード形式
- Again/Hard/Good/Easyボタン
- 進捗バー
- 残り枚数表示

### 5. クイズモード - 設定画面 (QuizSettings)
- 科目選択
- 出題形式（四択/入力式/穴埋め）
- 出題数（10/20/50/カスタム）
- 復習キュー優先ON/OFF

### 6. クイズモード - 問題画面 (QuizQuestion)
- 問題文
- 選択肢（四択）または入力欄
- 進捗表示

### 7. クイズモード - 結果画面 (QuizResult)
- 正答率
- 科目別内訳
- 苦手語リスト
- 間違いノート化ボタン

### 8. 設定画面 (Settings)
- 表示切替（英→日/日→英）
- 例文表示ON/OFF
- データエクスポート（Anki TSV）
- バックアップ/復元

## 主要ユーザーフロー

### 学習フロー
1. ホーム → 学習モード → 一覧
2. 一覧 → 用語タップ → 詳細
3. 詳細 → 関連語タップ → 別の詳細
4. 詳細 → 戻る → 一覧

### 復習フロー
1. ホーム → 今日の復習 → SRS復習
2. カード表示 → タップで回答表示
3. Again/Hard/Good/Easy選択
4. 次のカード → 完了

### クイズフロー
1. ホーム → クイズモード → 設定
2. 設定 → 開始 → 問題
3. 回答選択 → 正誤表示 → 次の問題
4. 全問完了 → 結果画面

## コンポーネント

### TermCard（一覧用）
- 左側：番号バッジ（丸形、青背景）
- 中央上：英語見出し（太字）+ 略語
- 中央下：日本語見出し（アクセントカラー）
- 右側：科目タグ
- 下部：薄い区切り線

### TermDetailView（詳細用）
- 見出し帯（薄いピンク背景）
- セクション区切り（細い線）
- 例文ブロック（インデント）
- Pitfall（赤枠）
- 関連語チップ（タップ可能）

### QuizOption（選択肢）
- 角丸ボタン
- 選択時：青枠
- 正解時：緑背景
- 不正解時：赤背景
